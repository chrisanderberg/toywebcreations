# Conway's Game of Life

**Conway's Game of Life** is a cellular automaton: a grid of cells that are either alive or dead, updating in discrete steps using simple neighbor rules.

## Rules

Each cell looks at its eight neighbors (here, the grid **wraps** at the edges like a torus, so nothing falls off the screen). Every generation:

- A live cell with 2 or 3 live neighbors stays alive.
- A live cell otherwise dies.
- A dead cell with exactly 3 live neighbors becomes alive.

## What this demo does

Paint patterns with the mouse or pointer, hit **Run** to watch them evolve, or **Step** one tick at a time. **Random** seeds noise; **Clear** wipes the board. If your system prefers reduced motion, continuous **Run** is disabled so the page stays calm—you can still **Step** manually.
